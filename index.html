Script 1: Admin Toolkit

Server Monitor & Real-Time Alert System

An essential Tampermonkey script for server administrators. Get eyes on your server 24/7 with immediate notifications and comprehensive activity logging directly integrated into the BattleMetrics interface.

  • Join/Leave Alerts: Set up custom audio alerts for specific players joining or leaving the server.
  • Historical Logging: Automatically track player activity and server population history for forensic analysis.
  • Seamless Integration: Runs quietly in the background, only displaying its dynamic panel when you need it.

Script 2: Player Research

Rust Analytics & Total Playtime Tracker

Transform the BattleMetrics player page into a powerful investigative dashboard. Get the exact, aggregated playtime and a detailed history of a player’s activity across all their top servers.

  • Accurate Total Playtime: Calculates and displays the player’s aggregated Rust playtime across the platform.
  • First Seen Date: Pinpoint exactly when a player first appeared on BattleMetrics.
  • Top Server Breakdown: View a categorized list of servers by the player’s total hours spent.
